**Hell-scape** is a 2D platformer created for the **EPITA S2 Project**. The game centers on the cooperation between two unlikely protagonists: **Angie**, an angel, and **Dimy**, a demon. Trapped in an infernal world inspired by Catholic and Greek mythologies, they must combine their unique powers to overcome obstacles and escape.
